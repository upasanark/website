ADVENT — Advanced Ventilation

Real-time respiratory monitoring for patients on mechanical ventilation using EMG signal analysis.

Overview

At GE Research, I designed and built a framework for analyzing high-frequency EMG signals to monitor and detect respiratory patterns in patients on mechanical ventilation. The system developed advanced signal processing algorithms for fast and accurate detection of patient breath signatures.

Key Contributions

  • Designed signal processing framework for high-frequency EMG analysis
  • Developed algorithms for fast and accurate detection of patient breath signatures
  • Implemented dynamic adaptive calibration to account for patient-to-patient variability
  • Built continuous monitoring and tuning system for real-time operation

Impact

Enabled continuous, accurate respiratory monitoring in critical care settings, reducing manual intervention and improving patient outcomes through adaptive, personalized signal analysis.

Skills & Tools

Signal processing, machine learning, real-time systems, adaptive algorithms, medical device development.
