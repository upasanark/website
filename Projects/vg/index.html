Vortex Generator

Machine learning-based inverse design framework for airfoil/blade performance prediction.

Overview

At GE Research, I conceptualized a machine learning-based inverse design framework and developed ML models for predicting airfoil/blade performance, reducing costs of wind tunnel tests and downtime by up to a week. The system enables rapid exploration of the design space by learning from historical DOE data and surfacing similar performance candidates.

Key Contributions

  • Built a dashboard with a lookup table of DOE points for performance-based search
  • Implemented ML models for predicting airfoil/blade performance from design parameters
  • Enabled visualization of small perturbations around selected designs
  • Reduced wind tunnel testing costs and design cycle downtime significantly

Impact

Cut wind tunnel testing costs and reduced design iteration time by up to a week by replacing physical testing cycles with ML-guided surrogate searches.
