# Sample Book Review

This is a sample to show the markdown workflow. Delete this file and create your own reviews here.

## How to Create a New Review

1. Create a new `.md` file in this folder (e.g., `sapiens.md`)
2. Add front matter at the top with `title`
3. Write in markdown — Jekyll renders it automatically
4. Commit and push — the page will appear on your site

## Example Front Matter

```yaml
---
title: "Book Title — Author Name"
---
```

That's it. The layout, nav, and styles are applied automatically.
